# 200DK peripheral sample-python

#### Directory structure and description

This directory contains examples of using various hardware interfaces with the Atlas200DK in python, using the [python-periphery](https://github.com/vsergeev/python-periphery) library. Each folder corresponds to examples of different hardware devices for users' reference. The directory structure and specific instructions are as follows.
| samples  | description  |
|---|---|
| [gpio](./gpio)  | Sample program to demonstrate driving a GPIO port on the Atlas 200 DK board using the python-periphery python library. |
| [i2c](./i2c)  | Sample program to demonstrate I2C connection between Arduino Uno and Atlas 200 DK board using the python-periphery python library.  |
| [uart](./uart)  | Sample program to demonstrate UART connection between Arduino Uno and Atlas 200 DK board using the python-periphery python library. |
