# 初级学习路径

#### 针对用户画像
有编程能力（c++/python），无昇腾CANN推理应用开发经验的开发者。

#### 前置技能要求
本课程需要具备以下前置技能。   
- 1.Liunx基础知识   
- 2.C++基础知识/python基础知识   
- 3.图像处理的基本知识（如：YUV/BGR等图像格式的了解）   
- 4.神经网络基础知识（pb/onnx模型概念）  
- 5.内存管理相关内容     

#### 期望达成目标
可以基于昇腾CANN独立开发一个CV类图片应用；对昇腾CANN有初步了解，可以使用ACL进行基础的应用开发。
